# AI Impact on Students - Data Analysis & Machine Learning

## Project Overview
This project analyzes the impact of Artificial Intelligence on students using Python. It demonstrates a complete data analysis workflow, from data preprocessing to machine learning model evaluation.

## Features
- Data Cleaning & Preprocessing
- Missing Value Handling
- Exploratory Data Analysis (EDA)
- Data Visualization
- Correlation Heatmap
- Feature Engineering
- One-Hot Encoding
- Feature Scaling
- Machine Learning Models
  - Linear Regression
  - Decision Tree Regressor
  - Random Forest Regressor
- Model Performance Evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Results
The project compares multiple machine learning models to predict outcomes and evaluates their performance using standard regression metrics.
